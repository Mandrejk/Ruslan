Macro Definition Documentation

#define ThreadManager_CreateThread (   pThreadType,
  pData 
)    Framework::classThreadInstance::_init_thread(new pThreadType(pData),__FILE__,__LINE__,__func__)

Макрос для создания потоков, вызывает функцию _init_thread, подставляя в нее имя файла, номер строки и имя функции исходного файла, в котором создается поток.

Пример создания потока:
class testClass: public Framework::classThreadInstance
{
testClass(void* pData) : classThreadInstance(pData) { ... }
}
int data;
ThreadManager_CreateThread(testClass,&data)

Parameters
[in]pThreadType- название класса потока. Если конструктор потока содержит переменные, они указываются.
[in]pData- указатель на пользовательские данные.
Returns
указатель на созданный поток
