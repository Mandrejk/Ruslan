Detailed Description

В файле содержатся макросы для настройки и использования модуля отладочной информации.

Для исполльзования макросов необходимо включить данный заголовочный файл и объявить модуль с помощью макроса MODULE_DECLARE.

При сохранении сообщений с отладочной информацией используются 2 условия:

  • Условия этапа компиляции. определяются макросами DBG_ERROR, DBG_WARNING, DBG_FUNCTION и DBG_TEST.
  • Условия выполнения приложения. Определяются зонами, которые могут настраиваться во время работы приложения. При отправлке отладочного сообщения, указываются зона (бит в 32-битной маске). Если зона включена для данного модуля, сообщение отправляется. Зоны по умолчанию указываются при объявлении модуля.

Если модуль реализован в нескольких файлах, объявлений может несколько, но зоны по умолчанию должны совпадать. Иначе, поведение не определено.
